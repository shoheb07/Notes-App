# Notes-App
Notes App (Python Tkinter) is a simple desktop text editor built using Python’s built-in Tkinter library. It allows users to create, edit, open, and save text files in a clean graphical interface.

A simple desktop Notes Application built using Python and Tkinter.
This app allows users to create, open, edit, and save text files easily.


🚀 Features

- Create new notes
- Open existing ".txt" files
- Save notes as ".txt"
- Simple and clean GUI
- Beginner-friendly code
- No external libraries required

🛠️ Technologies Used

- Python 3
- Tkinter (Built-in GUI library)

📦 Installation

1. Install Python (version 3.x recommended)
2. Clone this repository or download the project files

git clone https://github.com/your-username/notes-app.git
cd notes-app

▶️ How to Run

Run the following command:

python notes_app.py

The Notes App window will open.
